## Setup

1. Create a virtual environment (recommended):

    ```sh
    python -m venv .venv
    ```

2. Activate the virtual environment:

    - On Windows:
        ```sh
        .venv\Scripts\activate
        ```
    - On macOS/Linux:
        ```sh
        source .venv/bin/activate
        ```

3. Install dependencies:

    ```sh
    pip install -r requirements.txt
    ```
